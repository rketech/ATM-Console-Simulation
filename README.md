# ATM Console Simulation

A menu-driven ATM Console Simulation built in Kotlin.

This project was created as part of my Kotlin learning journey to practice programming fundamentals and object-oriented programming concepts.

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-Driven Interface
- Exit Option

## Concepts Used

- Variables and Data Types
- Functions
- Conditional Statements (if/else, when)
- Loops (while, do-while)
- Collections
- Null Safety
- Object-Oriented Programming

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run the Kotlin file containing the `main()` function.

## Learning Objective

The goal of this project is to gain hands-on experience with Kotlin programming by building a simple banking application that simulates basic ATM operations.
